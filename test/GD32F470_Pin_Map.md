# GD32F470 梁山派开发板引脚占用说明文档

本文档详细列出了当前 Keil 工程（CS1237、DAC 调试与 HLW8112 电能计量系统）中已被占用的 MCU 引脚及其功能用途，以防后续外设开发时发生引脚冲突。

## 各外设模块引脚详情说明

### 1. 调试串口 (USART0)
* **引脚**：PA9 (TX), PA10 (RX)
* **配置源文件**：`usart.c` / `usart.h`
* **说明**：配置为 AF7 复用功能，波特率通常初始化为 `115200`，供 `printf` 重定向输出。

### 2. 数模转换 (DAC)
* **引脚**：PA4 (通道0), PA5 (通道1)
* **配置源文件**：`bsp_dac.c` / `bsp_dac.h`
* **说明**：配置为模拟（Analog）通道，当前在 `main.c` 中已被注释掉，若后续需要使用板载 DAC 输出测试电压，请直接取消 `main.c` 中 `bsp_dac_init()` 等函数的注释。

### 3. 高精度模数转换 (CS1237 ADC)
* **引脚**：PC3 (SCLK), PC2 (DOUT)
* **配置源文件**：`bsp_cs1237.c` / `bsp_cs1237.h`
* **说明**：
  * **PC3 (SCLK)**：时钟输出引脚。由于 CS1237 芯片空闲时钟需保持为低电平且时钟高电平 $>100\mu\text{s}$ 会使芯片进入休眠，因此该引脚默认为 **低电平**。
  * **PC2 (DOUT)**：这是一个双向引脚，数据读取阶段作为**带上拉输入**；配置寄存器阶段会临时切换为**推挽输出**。

### 4. 状态指示灯 (LED2)
* **引脚**：PD7
* **配置源文件**：`main.c`
* **说明**：用于开发板板载的蓝色 LED 灯闪烁控制。

### 5. 电能电力计量 (HLW8112)
* **引脚**：PB11 (EN/RSTN), PB12 (SCSN), PB13 (SCLK), PB14 (SDO/MISO), PB15 (SDI/MOSI)
* **配置源文件**：`bsp_hlw8112.c` / `bsp_hlw8112.h`
* **说明**：
  * **PB11 (EN/RSTN)**：芯片的硬件使能引脚，拉高使能芯片，拉低会使芯片复位。
  * **PB12 (SCSN)**：片选引脚，低电平使能 SPI 通信。
  * **PB13 (SCLK)**：模拟 SPI 时钟脚，空闲状态（CPOL=0）保持低电平。
  * **PB14 (SDO/MISO)**：MCU 接收数据脚，设置为带上拉输入。
  * **PB15 (SDI/MOSI)**：MCU 发送数据脚，设置为推挽输出。

## 避坑与扩展指南

* **引脚复用注意**：
  在添加其他外设（如 I2C 或 PWM 脉宽调制）时，应避开以下已被占用的 12 个引脚：
  * **PA4, PA5, PA9, PA10**
  * **PC3, PC2**
  * **PD7**
  * **PB11, PB12, PB13, PB14, PB15**
* **VREF 与模拟地 (AGND) 注意**：
  在差分测量微小电压时，应保证 AIN- 与 AIN+ 外部线路远离高频数字信号线（如 PA9, PA10 串口线和 PC1 时钟线），并确保 GND 接触良好，以防引入高频噪声干扰 ADC 采样。
